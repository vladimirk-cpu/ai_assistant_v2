# AI Assistant — локальный AI-агент для Windows

## Быстрый старт

1. Запустите `ai-assistant.exe` (или установите через установщик).
2. Откроется консольное окно — не закрывайте его (это сервер).
3. В любом OpenAI-совместимом клиенте (Roo Code, Continue, Cursor) настройте:
   - **Base URL:** `http://localhost:8000/`
   - **API Key:** любой текст (например, `local`)
4. Начните чат!

## Как это работает

- **Architect** — проектирует архитектуру.
- **Code** — пишет код.
- **Debug** — отлаживает (по запросу).
- **Review** — ревьюит код (по запросу).

Агенты сами решают, участвовать или пропустить ход, в зависимости от задачи.

## Параметр `agents` (выборочный запуск)

Можно отправить запрос только к определённым агентам:

```json
{
  "messages": [{"role": "user", "content": "..."}],
  "agents": ["Architect", "Code"]
}
```

## Где хранятся данные
Memory Bank (контекст): `%APPDATA%\AI-Assistant\memory-bank\`

Логи: `%APPDATA%\AI-Assistant\logs\app.log`

## Безопасность
Запрещены опасные команды (`rm -rf`, `del /f /s` и т.д.).

При попытке выполнить опасную команду агент запрашивает подтверждение (в консоли сервера).

## Остановка сервера
Нажмите `Ctrl+C` в окне консоли, где запущен `ai-assistant.exe`.

## Установка (опционально)
Запустите `AI_Assistant_Setup.exe`, следуйте инструкциям. После установки ярлык появится на рабочем столе и в меню «Пуск».
